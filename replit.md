# Maison

A luxury e-commerce web app — silk, cashmere, leather and gold — built with **TanStack Start** (SSR), **React 19**, **Supabase**, **Razorpay** (payments), and **Twilio** (SMS). The brand is "Maison", described as an editorial luxury atelier from Mumbai.

## Stack

| Layer | Technology |
|---|---|
| Framework | TanStack Start (SSR) + TanStack Router (file-based) |
| UI | React 19 + Tailwind CSS v4 + shadcn/ui (Radix) |
| Backend / Auth | Supabase |
| Payments | Razorpay |
| SMS | Twilio |
| Build | Vite 8 + Bun |

## How to run

```bash
bun install        # install dependencies (first time only)
bun run dev        # starts the dev server on port 5000
```

The Replit workflow **Start application** runs `bun run dev --port 5000 --host 0.0.0.0` automatically.

## Environment variables

Sensitive credentials are stored as **Replit Secrets** (never committed). Non-sensitive values (`SUPABASE_URL`, `TWILIO_FROM_NUMBER`, etc.) live in `.env`, which is git-ignored. Use `.env.example` as a template — copy it to `.env` locally and fill in real values; add secrets via Replit's Secrets panel.

Razorpay keys are still placeholders — replace them via Replit Secrets before going live.

| Variable | Purpose |
|---|---|
| `SUPABASE_URL` / `VITE_SUPABASE_URL` | Supabase project URL |
| `SUPABASE_PUBLISHABLE_KEY` / `VITE_SUPABASE_PUBLISHABLE_KEY` | Supabase anon/publishable key |
| `RAZORPAY_KEY_ID` / `RAZORPAY_KEY_SECRET` / `RAZORPAY_WEBHOOK_SECRET` | Razorpay payment gateway |
| `TWILIO_ACCOUNT_SID` / `TWILIO_AUTH_TOKEN` | Twilio SMS credentials |
| `TWILIO_FROM_NUMBER` / `TWILIO_MESSAGING_SERVICE_SID` | Twilio sender config |

## Routes

File-based routing under `src/routes/`. Key routes:

- `/` — homepage with hero + category grid
- `/shop` — all products
- `/product/$slug` — product detail page
- `/cart` — shopping cart
- `/sell` — seller portal
- `/plans` — subscription plans
- `/_authenticated/*` — auth-protected pages
